# timebalance
TimeBalance adalah aplikasi manajemen waktu yang membantu pengguna meningkatkan produktivitas melalui sistem Work–Reward Mechanism. Pengguna harus menyelesaikan tugas terlebih dahulu untuk mendapatkan waktu hiburan sebagai reward, sehingga membantu mengurangi prokrastinasi, meningkatkan fokus, dan menciptakan keseimbangan hidup
